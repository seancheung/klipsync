8 个页面覆盖完整流程

本页是静态原型的入口索引。每个页面都可直接在浏览器打开、独立验收。核心是 P-004 主工作台，其他页面围绕它展开。

P-004 · 核心页面

主工作台

双栏布局：左剪贴板列表 + 右编辑器。含工具栏、附件网格、二维码面板、清空/删除二次确认。
核心 多端同步
P-001

首次部署引导

系统无管理员时强制落地，创建 Admin 账号后自动登录进入工作台。
一次性
P-002

登录页

用户名 + 密码 + 记住我。扫码进入但未登录时也会落到这里。
认证
P-003

强制改密

初始密码或管理员重置后的密码必须在首次登录立即修改。除登出外无其他跳出路径。
must_reset
P-005

个人设置

账号信息、修改密码、登出。管理员还能在此跳入管理面板。
自助
P-006 · 仅 Admin

用户管理

表格 + 新增用户 + 重置密码 + 删除用户（需输入用户名二次确认）。无法查看任何剪贴板内容。
Admin only
P-007 · 仅 Admin

系统设置

附件大小上限配置、存储概览、版本信息。
Admin only
P-008

无权访问状态页

扫码越权、URL 失效、对象已删除时的软落地页。
错误兜底
