设计原则

06 / Principles

画布是纯白，不是灰

window 外层 #F5F5F7，编辑区 / 卡片 #FFFFFF。靠纯白内容 + 浅灰 frame 建立"这是一张作品台"的层次，不靠花哨阴影。

紫色只在"主动作"和"信号"上

填充紫：Logo / Primary CTA（登录、创建、确认）。Pastel 紫徽章：置顶、"已同步"反馈。正文 / 表单中不出现紫色。

Pastel 徽章是情绪，不是装饰

状态胶囊用粉彩（桃色 / 薄荷 / 玫红 / 琥珀 / 天蓝 / 紫），每个颜色都有唯一语义：上传中 / 已保存 / 已删除 / 保存中 / 提示 / 置顶。不随意复用。

工具气质靠选择性 mono

时间戳、剪贴板 ID、URL、文件大小、快捷键提示用 Geist Mono；其他一律 Geist / Inter。mono 过多会把"工具"变成"终端"。
